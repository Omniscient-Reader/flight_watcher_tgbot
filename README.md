# 🌪 Aviasales Flight Tracker Bot

Бұл — Aviasales жүйесінен әуе билеттерінің бағасын бақылап, қолданушы орнатқан шекті бағаға жеткенде немесе одан арзандағанда Telegram арқылы автоматты түрде хабарлама жіберетін робот (Бот).

## ✨ Ерекшеліктері

* 🛫 **Ыңғайлы инлайн-мәзір:** Бағыттарды, ұшу күнін және күтілетін бағаны толық батырмалар арқылы реттеу.
* 📅 **Форматты тексеру (Validation):** Күн дұрыс енгізілмесе (YYYY-MM-DD), бот қателік жібермей, қайта сұрайды.
* 📋 **Рейстерді басқару (/collections):** Бақылауға алынған барлық рейстерді көру және керек емес рейсті бір батырмамен өшіру.
* 🔒 **Қалып қоюдан қорғаныс:** Батырмаларды бейберекет баса бергенде боттың қатып қалуынан толық қорғалған.
* 🛑 **Дубликаттан қорғау:** Бір рейсті базаға екі рет қайталап жазуға жол бермейді.

---

## 📂 Жобаның құрылымы

* `bot.py` — Боттың негізгі логикасы, командалар мен инлайн батырмалар өңдеушісі.
* `database.py` — PostgreSQL (немесе басқа) деректер базасымен байланыс модулі.
* `parser.py` — Aviasales API-імен немесе парсерімен жұмыс істеп, билет бағасын алу.
* `config.py` — Жобаның жалпы баптаулары мен тұрақты мәндері.
* `requirements.txt` — Ботты іске қосуға қажетті Python кітапханаларының тізімі.

---

## 🛠 Іске қосу нұсқаулығы

### 1. Репозиторийді көшіріп алу және папкаға өту:

```bash
git clone https://github.com/сенің-юзернеймің/Omniscient-Reader.git
cd Omniscient-Reader
```

### 2. Қажетті кітапханаларды орнату:

```bash
pip install -r requirements.txt
```

### 3. Конфигурацияны баптау:

Жобаның түбірінде `.env` файлын жасап, ішіне өз мәліметтеріңізді жазыңыз:

```env
BOT_TOKEN=сенің_телеграм_бот_токенің

# Егер база қолданылса, база параметрлерін де осында жазасыз
DB_NAME=flight_tracker
DB_USER=postgres
DB_PASSWORD=password
DB_HOST=localhost
DB_PORT=5432
```

⚠️ **Маңызды:** `.env` файлы құпия ақпаратты сақтайтындықтан, ол `.gitignore` файлына қосылған және GitHub-қа жүктелмейді!

### 4. Ботты іске қосу:

```bash
python3 bot.py
```

## 🤖 Боттың Командалары

### `/start` немесе `/filter`

Іздеу сүзгісін ашу:

- Қайдан
- Қайда
- Күні
- Бағасы

### `/collections`

Қазіргі таңда бақылауда тұрған барлық рейстерді тізімдеу және өшіру.

---

## 🔄 Жұмыс логикасы

1. Қолданушы `/filter` командасын ашады.
2. Ұшу бағытын, күнін және күтілетін бағаны енгізеді.
3. Рейс деректер базасына сақталады.
4. Бот билет бағаларын автоматты түрде тексеріп отырады.
5. Баға қолданушы көрсеткен шекке жеткенде Telegram арқылы хабарлама жібереді.
6. Хабарламадағы батырма арқылы билет сатып алу бетіне өтуге болады.
